# Focus Mode Activator

Chrome extension (MV3) — silence browser notifications across all tabs with one click.

**Version:** `0.4.0`

## Quick start

1. Load unpacked at `chrome://extensions` → select this folder
2. Open popup → **Open notification test page**
3. **Enter Focus Mode** → test → **Exit Focus Mode** for session report
